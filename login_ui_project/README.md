 # Flutter Login User-interface :

This project is a simple **Flutter user interface application** developed as part of my Flutter learning and internship practice. The application demonstrates how to build a clean and functional login screen using basic Flutter widgets and simple navigation between screens.

The goal of this project is to practice building UI layouts, handling user input, and implementing basic form validation in Flutter.

## Features

* Login screen with **Email and Password fields**
* **Form validation** for email format and password input
* Login button for submitting the form
* "Forgot Password?" text option
* Navigation from the **Login Screen to the Home Screen**
* Clean layout using common Flutter widgets

## Widgets Used

This project uses some of the core Flutter widgets including:

* Column
* Row
* Container
* Text
* TextFormField
* TextButton
* Scaffold
* Navigator

These widgets help structure the UI and create a simple and responsive layout.

## Validation

The login form includes basic validation:

* Email field checks for a **valid email format**
* Password field ensures that the **input is not empty**

## Navigation

The application demonstrates **screen navigation in Flutter** using:

Navigator.push()

This allows users to move from the login screen to the home screen after successful validation.

## Technologies Used

* Flutter

* Material Design Widgets

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/usmankhanxo5/flutter-login-ui.git
```

2. Navigate to the project folder

```
cd flutter-login-ui
```

3. Install dependencies

```
flutter pub get
```

4. Run the application

```
flutter run
```
